# control_car

## 项目简介

为电赛控制题设计的一套硬件与软件，其中包括:

- 实现 pin2pin 兼容的 stm32 与 mspm0 核心板两块
- 集成底板一块以及对应的驱动代码
- 各类常用传感器的驱动代码及运动控制代码
- 中间层实现对 stm32 库与 mspm0 库的兼容

## 软件架构

```
User
├─ Device    放置LED、蜂鸣器、按键等设备
├─ It        放置中断函数，定时器中断、uart中断等
├─ Middle    中间转接、转接stm32与mspm0的驱动库
├─ Motor     各种电机的驱动，直流电机、舵机、步进电机
├─ Sensor    传感器驱动
├─ Status    状态树
└─ Tool      常用工具(数学宏函数、PID等)
```

## 项目特点

- 传感器的读取与解析分开(详见 sensor 的 README)
- 状态树集成(详见 status 的 README)

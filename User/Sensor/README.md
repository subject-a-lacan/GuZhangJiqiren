# 项目传感器读取部分介绍

## 设计特点

这个项目的传感器读取是与常见的小车工程有很大区别的地方。最大的不同是：将传感器原始数据的读取与对原始数据的解析分开进行。

## 优势

这样处理的好处是：所有传感器的原始数据可以使用 DMA 进行读取，不需要占用 CPU 资源，可以无阻塞的同时读取多个传感器。

## 实现方式

以 gy901 陀螺仪为例，传感器根据需要分为三类函数：

- 初始化函数：`init_gyr(GYR *gyr)`
- 获取原始数据函数：`void get_gyr_data(GYR *gyr);`
- 解析原始数据函数：`float get_gyr_value(GYR *gyr, enum gyroscope key);`

## 工作流程

获取原始数据函数放置在 `status_update()` 中，解析原始数据函数放置在 `status_driver()` 中。
`status_update()` 与 `status_driver()` 每 25ms 交替调用实现对数据的读取与解析。

```
时间线:
0ms     ->     25ms     ->    50ms     ->    75ms         ...
获取原始数据    解析原始数据    获取原始数据    解析原始数据 ...
```

# 第一问路线说明

## 场地布局

```
A (左上角) ●═══════════● B (右上角)
           ║           ║
           ║  逆时针   ║
           ║           ║
D (左下角) ●═══════════● C (右下角)
```

- 正方形外边长 100cm × 100cm
- 黑色胶带宽 1.8cm ± 0.2cm
- 四个顶点逆时针：A → D → C → B → A

## AB 发车点

- **车身位置**：车身与 AB 边对齐（车身长轴在 AB 横线上）
- **车头朝向**：朝左，指向 A 点
- **前沿位置**：小车前沿中心点投影在 AD 线外沿上
- **行驶方向**：逆时针

## 行驶路径

路径 = A → D → C → B → A（逆时针一整圈）

```
出发姿态：
        车头← ██
A ●═══════════════● B
  │               │
  ↓               ↑
  │               │
  │               │
D ●═══════════→ ● C
```

### 四个拐角（全是左转）

| 拐角序号 | 位置 | 动作 | cross_cnt |
|---|---|---|---|
| 拐角 1 | A 点 | 左转，车头从朝左变为朝下，驶入 AD 边 | 0→1 |
| 拐角 2 | D 点 | 左转，车头从朝下变为朝右，驶入 DC 边 | 1→2 |
| 拐角 3 | C 点 | 左转，车头从朝右变为朝上，驶入 CB 边 | 2→3 |
| 拐角 4 | B 点 | 左转，车头从朝上变为朝左，驶入 BA 边 | 3→4 |

### 停车段

拐角 4（B 点）转弯完成后：
- 车在 BA 边上，车头朝左（朝向 A）
- 沿 BA 边巡线直走约 100cm 回到 A 点
- **到达 A 后不转弯，直接停车**

## 停车姿态

```
停车时：
        车头← ██
A ●═══════════════● B
  ↑ AD 线
  │
  │ 前沿中心在 AD 线外沿上
  │ 或在外沿后方 ≤ 3cm 处
  │
D ●
```

- 车身与 AB 对齐
- 车头朝 A（朝左）
- 前沿中心不得超过 AD 线外沿 3cm

## 关键约束

- 全程时间 ≤ 15 秒
- 停车偏差 D1 ≤ 3 cm
- 启动和停车需声光提示
- 行驶过程中车身投影不得完全脱离轨迹线

## 两个概念：启动死区 vs 稳态截距

- **启动阈值 = 253 CCR**：从静止到轮子开始转，需要克服静摩擦，PWM 必须 >= 254。实测 PWM=253 时轮子不动。
- **稳态截距 = 157 CCR**：用已经稳定运动的数据做线性回归，直线往 speed=0 方向外推得到的截距。它小于启动阈值，因为动摩擦 < 静摩擦——轮子一旦转起来，维持同样速度需要的 PWM 比启动时小。

两者不是同一个东西。

## 线性拟合（稳态）

```
PWM = steady_offset_pwm + kff * speed
```

| 参数 | CCR 值 | PWM 百分比 |
|------|--------|-----------|
| steady_offset_pwm (稳态截距) | **157** | 5.2% |
| kff (斜率) | **18.3** | 0.61%/pulse |
| start_min_pwm (启动下限) | **254** | 8.5% |

平均绝对误差 = 4.9 CCR，R² = 0.999。

启动阈值 253 仅作为低速下限保护使用，不参与稳态拟合。

## 前馈公式

```
if (tar_speed == 0) {
    ff = 0;
} else {
    ff_abs = 157.0f + 18.3f * fabsf(tar_speed);
    if (ff_abs < 254.0f) {
        ff_abs = 254.0f;   // 低于启动死区则抬到 254，保证能从静止推动
    }
    ff = sign(tar_speed) * ff_abs;
}
```

常用速度段 (30~55 pulse/20ms) 下 ff 天然 > 254，下限保护只在 v < 5.3 时触发。
